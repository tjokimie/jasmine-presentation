suites & specs

desribe('Super cool calculator', function () {
    var counter = 1;

    it('should add a number', function () {
        counter = counter + 2; // counter is 1
        expect(counter).toBe(3);
    });

    it('should multiply a number', function () {
        counter = counter * 5; // counter is 3!
        expect(counter).toBe(15);
    });
});
  • Test suite alkaa globaalin describe-funktion kutsulla
    • Ensimmäinen string-parametri kuvaa mitä on testin suorituksen alla
    • ...toinen parametri, funktio, implementoi suiten
  • Test spec alkaa globaalin it-funktion kutsulla
    • Ensimmäinen string-parametri on specin otsikko
    • ...toinen parametri, funktio, sisältää specin, eli itse testin
      • Spec sisältää yhden tai useamman assertin suorituksen alla olevan koodin tilasta
